The Rexel UX Team has been set up with Amazon Web Services (AWS). To read more about the overall structure of this wiki, visit the wiki article.

The services currently integrated include:
  • Github integration with CodeDeploy via webhooks
  • CodeDeploy for deployment to ec2 instances
  • 3 instances of ec2 servers
  • s3 Storage for UI assets and private storage
  • Route53 for domain name purchasing and url routing to ec2 server instances.


There is a way to send notifications when code is deployed via CodeDeploy. I believe the service is called 'Lambda'. This is a feature I would like to configure in the near future. A recommendation has also been made to move our main ecommerce servers to AWS and automate the build/deployment process. This is still in the works and may require much more persuasion.
